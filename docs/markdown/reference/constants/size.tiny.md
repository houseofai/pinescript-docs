### size.tiny

A constant to specify the size of the graphics drawn by [plotchar](#fun_plotchar), [plotshape](#fun_plotshape), [label.new](#fun_label.new), [box.new](#fun_box.new), and [table.cell](#fun_table.cell). Sets the size to tiny.

Type

const string
