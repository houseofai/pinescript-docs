### size.auto

A constant to specify the size of the graphics drawn by [plotchar](#fun_plotchar), [plotshape](#fun_plotshape), [label.new](#fun_label.new), and [box.new](#fun_box.new). Adjusts the size of the graphics automatically.

Type

const string
